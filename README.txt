各ファイル好きなようにカスタマイズして使用してください。
ページを装飾するCSSは設定していません。
クラス名は指定してありますが、変えて頂いて構いません。

オリジナルテーマを作成する際の最低限のファイル等は
入れてあると思うのでご自由に使用してください。

今回使用しているコード以外の記述はこちらに載せてあります。
（https://note.com/toom_design/n/n43ae684c6287）


以下、説明

front-page.php
----------------------
トップページとなるファイル


404.php
----------------------
404となるファイル


footer.php 、header.php、sidebar.php
----------------------
フッター、ヘッダー、サイドバーの記述をするファイル


breadcrumb.php
----------------------
パンくずに使用する為の記述が書いてあります。


functions.php
----------------------
初期であると良いかな程度の記述がしてあります。
適宜自由に追加してください。


style.css
----------------------
テーマ情報を書く場所です。


img
----------------------
使用する画像をいれてください


js
----------------------
使用するJSファイルをいれてください


scss
----------------------
使用するscssファイルをいれてください
※初期設定的な部分は記述してあります。適宜追加してください。
